# TemperFine
## About
--------

TemperFine is a 3D voxel-based multiplayer real-time strategy game which takes place on an alien world.

Written in C++ with raw OpenGL 4.0.

## System Requirements
----------------------

* Intel Atom Z3735F
* OpenGL 4.0.0 compatible GPU
* 2 GB RAM
